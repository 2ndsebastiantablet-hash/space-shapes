# DROP//CHAIN

Fast arcade survival and chain-reaction game built as a single HTML Canvas prototype.

## Gameplay Systems

Collect ancient animal statue fragments during a run to complete a Stand. Completing Lion, Owl, Snake, Jackal, or Crocodile fragments transforms the player for 60 seconds with a unique attack style and arena effect.

## Hosting

This repo is ready for static web hosting. Serve the repository root and open `index.html`.

Short gameplay sound effects are generated with the Web Audio API. The game does not include background music.
